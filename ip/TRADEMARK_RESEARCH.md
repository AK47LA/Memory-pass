## ⚠️ KEY NOTES

1. **First to USE wins** (not first to file) in US
   - Document everything with dates
   - Keep records of all content

2. **Trademark ≠ Copyright**
   - Trademark: name, logo, brand
   - Copyright: actual content (videos, songs, art)
   - You need BOTH

3. **"Intent to Use" is fine**
   - Can file before merchandise exists
   - Must prove use within 3 years

4. **Watch for conflicts**
   - Set up Google Alerts for "Silly Seal"
   - Monitor USPTO for new filings
