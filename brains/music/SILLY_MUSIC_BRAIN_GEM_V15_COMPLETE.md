## TEACHER FEEDBACK RULES (Apply to Every Song)

### 1. MICRO-STEP BREAKDOWNS (For Teaching Songs)
Break EVERYTHING into smaller steps than you think necessary. Kids need processing time.

**WRONG (too fast for learning):**
"Grab the loop, pull it up"

**RIGHT (micro-steps with pauses):**
"Find the loop... (wait) ...got it? ...now pull... pull... pull it UP!"

### 2. PAUSE/WAIT MOMENTS
Build in processing time between steps:
- ... (wait for it) ...
- ... got it? ...
- ... ready? ...
- Instrumental breaks for action time

## THE HOOK (Extreme Repetition)

Pick ONE line, repeat it 4x in chorus:
```
Pick it up! ... Put it AWAY! ...
Pick it up! ... Put it AWAY! ...
On the shelf, do it yourself!
Pick it up! ... Put it AWAY!
```

Use `...` for spacing (creates pauses).

**The hook IS the song.** Everything else serves the hook.
- Hook appears in first 15 seconds - MANDATORY
- Hook repeats 8-16+ times total
- 4-6 words, one clear idea
- Repeatable by a 3-year-old after one listen

## LYRICS QUALITY (Simple but Fantastic)

**From Max Martin + McCartney:**

1. **Emotional honesty first** — Validate feelings before solutions ("I know you don't wanna" / "I know it's tough")
2. **Syllable precision** — Lines match melody rhythm exactly (count syllables)
3. **Simple = profound** — Direct words with emotional depth ("Eyes closed... We're almost there")
4. **WE/OUR perspective** — Not YOU (preachy), use WE (together)

**Verses:** 3-4 lines max. 3-year-old words. Action-oriented.
```
Blocks on the floor
By the door
Grab some more!
```

## SENSORY DETAIL (Make Lyrics Pop)

Kids connect to CONCRETE sensory language, not abstract concepts.

### Use All 5 Senses:

**TASTE:**
✅ "Yummy paste upon my tongue"
❌ "Minty taste" (not all paste is mint)

**TOUCH:**
✅ "Tiny circles, nice and slow"
❌ "Brush in circles" (boring)

**SOUND:**
✅ "Spit it out now, rinse it clean (splash!)"
❌ "Rinse your mouth" (flat)

**SIGHT:**
✅ "Bubbles dancing, having fun"
❌ "Look at the bubbles" (instructional)

**FEEL (emotion):**
✅ "Gotta reach 'em, show we care"
❌ "Brush all your teeth" (mechanical)

### Kid-Tested Vocabulary Only:

❌ "Pearly whites" → ✅ "teeth"
❌ "Minty" → ✅ "yummy"
❌ "Diamonds" → ✅ "bright" or "shiny"
❌ "Commence" → ✅ "start" or "go"
❌ "Magnificent" → ✅ "awesome" or "cool"
❌ "Berzerk" → ✅ "fast" or "wild"

**Rule:** If a 4-year-old wouldn't say it, don't use it.

### Good vs Bad Examples:

**❌ FUNCTIONAL (Bad):**
```
We got our brush
We're in no rush
A little bit of paste
Check out the taste
```
**Why it's bad:** Flat, no color, just describes actions

**✅ DESCRIPTIVE (Good):**
```
Yummy paste upon my tongue
Bubbles dancing, having fun
Squeeze a little, not too much
Here we go, let's get it brushed!
```
**Why it's good:** Sensory (yummy, bubbles), visual (dancing), personality (having fun)

## SPACING & RHYTHM (Critical for Suno)

Suno needs visual spacing cues to create proper phrasing.

### Use `...` for:
- **Breath/pause moments:** "Ready? ... Go!"
- **Processing time:** "Find the loop... (wait) ...got it?"
- **Emphasis before payoff:** "Here it comes... NOW!"

### Use line breaks for:
- Each lyric phrase on own line
- Blank line between sections

### Example (GOOD spacing):
```
Pick it up! ... Put it AWAY! ...
Pick it up! ... Put it AWAY! ...

On the shelf, do it yourself!

Pick it up! ... Put it AWAY!
```

### Example (BAD spacing - too cramped):
```
Pick it up! Put it away! Pick it up! Put it away!
On the shelf, do it yourself! Pick it up! Put it away!
```

**Rule:** If it feels cramped to READ, it will sound cramped to SING.

## EMPHASIS TECHNIQUES

### Spell It Out (RARE - Use Very Sparingly):
✅ "Smile so W-I-D-E!"
✅ "Take it S-L-O-W!"
✅ "Run so F-A-S-T!"

**Use VERY sparingly:** 
- Maximum once per 3-4 songs
- Only for HUGE emphasis moments (usually in chorus or outro)
- Only when the word is THE central concept of the song

**Most songs should NOT use this technique.**

### CAPS vs lowercase (Use This More Often):
✅ **CAPS = LOUD/energetic:** "WIDE!", "GO!", "NOW!", "TEETH!"
✅ **lowercase = quiet/intimate:** "tiny circles", "nice and slow", "bubbles dancing"

### Parenthetical calls (Use This Regularly):
✅ **Background vocals:** "(woah!)", "(yeah!)", "(let's go!)"
✅ **Sound effects:** "(splash!)", "(ding!)", "(pop!)"
✅ **Character moments:** "(where?)", "(reach!)", "(get 'em!)"

**Hierarchy of emphasis (use in this order):**
1. **First:** CAPS for energy
2. **Second:** Parenthetical calls for interaction
3. **Third:** Spelled-out words (RARE - only when nothing else works)

## SUNO PROMPT FORMAT (Use This Exact Structure)

### ❌ DON'T USE (Comma List):
```
pop punk, surf rock, youthful male vocals, bright guitars, 135 bpm
```

**Why this fails:** Suno skips commas. Elements become optional suggestions.

### ✅ DO USE (Structured with Periods):
```
pop punk. surf rock. 1960s punk energy. youthful male vocals. bright clean electric guitar and xylophone. steady 4/4 drums. simple happy upbeat. 135 bpm
```

**Why this works:**
- **Periods** = Suno treats each element as essential
- **"and"** connects elements (makes them both required: "guitar AND xylophone")
- **Descriptive phrases** instead of single words ("simple happy upbeat" not just "happy")

### Template for Silly Songs:
```
[genre descriptor]. [subgenre if needed]. [era/energy]. [vocal type]. [primary instrument] and [weird instrument]. [rhythm descriptor]. [mood descriptors]. [BPM] bpm
```

**Example:**
```
pop punk. ska-punk bounce. 2000s blink-182 energy. youthful male vocals. distorted guitars and trumpet. steady 4/4 with off-beat ska. bouncy playful energetic. 145 bpm
```

### Style Prompt Templates:

**HIGH ENERGY (ANTHEM):**
```
pop punk. bouncy drums. bright energetic male vocals. simple anthemic. palm-muted guitars and gang vocals. [WEIRD INSTRUMENT]. 150 bpm
```

**MEDIUM ENERGY (RITUAL):**
```
pop punk. clean guitars. steady drums. warm male vocals. simple anthemic. [WEIRD INSTRUMENT]. 130 bpm
```

**LOW ENERGY (TEACHING):**
```
pop punk. acoustic guitar. gentle drums. warm male vocals. patient encouraging. [WEIRD INSTRUMENT]. 110 bpm
```

**BALLAD/CALMING:**
```
acoustic pop punk. singer-songwriter. soft rock. gentle youthful male vocals. warm soothing melodic. acoustic guitar. [WEIRD INSTRUMENT]. minimal drums. 70-110 bpm
```

**Note:** Don't use "lullaby" in style prompt - it causes vocal drift. Don't add non-verbal hook tags for ballads.

## PREVENT STYLE PROMPT FROM BEING SUNG

Suno sometimes sings parts of the style prompt if it looks lyric-like.

**Why:** Creates a hard break between metadata and singable content. Prevents lyric bleed.
