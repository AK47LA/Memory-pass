### STEP 3: UPLOAD RIFF TO SUNO

**In Suno interface:**

1. Click **"Upload Audio"** (bottom left)
2. Select your acoustic riff file
3. Suno analyzes it → gives you continuation options

**Two paths from here:**

---

### PATH A: EXTEND FROM RIFF (Let Suno Build)

**Use when:** Riff is good, want Suno to finish it

1. Click **"Extend"** on uploaded riff
2. Add Suno Brain prompt (see Step 4)
3. Suno builds song around your riff

**Pro:** Fast, keeps your riff intact  
**Con:** Less lyric control

---

### PATH B: USE RIFF AS REFERENCE (More Control)

**Use when:** Want specific lyrics, full control

1. Upload riff → note the BPM Suno detects
2. Write full lyrics using Suno Brain (see Step 4)
3. Paste lyrics + style prompt
4. Set "Reference Track" = your riff
5. Generate

**Pro:** Full lyric control, structured properly  
**Con:** Might drift from original riff sound

---

### STEP 4: SUNO BRAIN PROMPT FORMAT

**Tell Suno Brain (Gemini Gem):**

```
Write a [SONG TYPE] song for Silly the Seal about [PAIN POINT].

The riff I uploaded is [ACOUSTIC/ELECTRIC] and [TEMPO].

I need:
- Full lyrics (WE/OUR perspective, 4-year-old words)
- Hook that repeats 8-16x
- One weird instrument: [YOUR CHOICE or let it pick]
- Suno-formatted style prompt
- Under 2 minutes
```

**Suno Brain will output:**

1. **Full lyrics** with Suno structure tags:
   ```
   [Intro]
   [Chorus]
   [Verse 1]
   etc.
   ```

2. **Style prompt** (paste into Suno):
   ```
   pop punk. acoustic guitar. youthful male vocals. warm melodic. [WEIRD INSTRUMENT]. 130 bpm
   ```

---

### STEP 5: PASTE INTO SUNO

**In Suno:**

1. **Custom mode** (toggle on)
2. **Lyrics box:** Paste Suno Brain lyrics (with [tags])
3. **Style of Music box:** Paste style prompt
4. **Reference audio:** (optional) Upload your riff again if using Path B
5. **Generate**

## THE STRUCTURE YOU NEED

**Suno Brain handles this, but here's what it creates:**

### Simple Structure (Most Silly Songs):
```
[Intro - Weird Instrument Solo]
(Grab your brush!)

[Chorus]
Hook line 1! ... Hook line 2! ...
Hook line 1! ... Hook line 2! ...
Payoff line!

[Verse 1]
Short line
Short line
Short line
Short line!

[Chorus]

[Verse 2]

[Chorus - Final Push]

[Outro]
```

**Key rules:**
- Hook in first 15 seconds (MANDATORY)
- Under 2 minutes total
- WE/OUR perspective (not YOU)
- 4-year-old vocabulary only
- One weird instrument called out
